## Getting Started

```bash
$ npm install

$ npm run start:dev
또는 
$ npm run start
```

만약 이후 실행이 되지 않는다면 
```bash
$ npm install --force
```
후 재실행 부탁드립니다.


-------------------------


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
